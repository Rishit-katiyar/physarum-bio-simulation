## Introduction

Physarum polycephalum is a species of slime mold that has garnered attention due to its remarkable abilities to navigate mazes, solve complex optimization problems, and exhibit emergent behaviors. This project seeks to create a computational model that mimics the behavior of Physarum polycephalum, providing insights into its underlying mechanisms and potential applications in various fields.

## Biological Behavior

### Amoeboid Movement

Physarum polycephalum exhibits amoeboid movement, a mode of locomotion characterized by the extension and retraction of cytoplasmic pseudopodia. These pseudopodia enable Physarum to crawl across surfaces and navigate through its environment. Amoeboid movement is facilitated by cytoplasmic streaming, a process in which the cytoplasm flows in a coordinated manner, propelling the organism forward.

### Chemotaxis

Physarum is capable of chemotaxis, the directed movement of an organism in response to chemical gradients in its environment. It can detect and respond to various chemical cues, such as nutrients, repellents, and pheromones, by altering its direction of movement. Chemotaxis allows Physarum to efficiently locate and navigate towards sources of nutrients, optimizing its foraging behavior.

### Foraging Behavior

Physarum exhibits sophisticated foraging behavior, dynamically adjusting its path to optimize the acquisition of nutrients in its environment. It explores its surroundings by extending pseudopodia in search of food sources, such as bacteria, fungi, and decaying organic matter. Through a process of trial and error, Physarum identifies and exploits spatial gradients of nutrients, maximizing its foraging efficiency over time.

### Network Formation

Physarum is capable of forming intricate networks of interconnected tubes, known as protoplasmic veins, which facilitate nutrient transport and information exchange throughout its body. These networks exhibit emergent properties, such as adaptive routing and self-repair, which contribute to the resilience of the organism. Physarum dynamically adjusts the formation of its network structures in response to changes in environmental conditions, optimizing resource allocation and distribution.

### Problem-Solving Abilities

Physarum has demonstrated remarkable problem-solving abilities in laboratory experiments, such as maze navigation, shortest path finding, and optimization tasks. It achieves these feats through the decentralized integration of environmental cues and the adaptive formation of network structures. By exploiting the inherent parallelism and redundancy of its decentralized network architecture, Physarum efficiently solves complex problems, often outperforming conventional algorithms in terms of efficiency and robustness.

## Installation

To run the Physarum bio-simulation, follow these steps:

1. Clone the repository to your local machine:
    ```bash
    git clone https://github.com/Rishit-katiyar/physarum-bio-simulation.git
    ```

2. Navigate to the project directory:
    ```bash
    cd physarum-bio-simulation
    ```

3. Install the required dependencies using pip:
    ```bash
    pip install -r requirements.txt
    ```

4. You're all set to run the simulation!

## Usage

To run the Physarum bio-simulation, execute the `main.py` script:

```bash
python main.py
```

This will initiate the simulation, and you'll be able to observe the behavior of the simulated Physarum polycephalum. Feel free to modify the parameters and experiment with different settings to explore the behavior of the organism further.
